# mohs_surgeons
Running the get_surgeon_demographic_data.R script will create all necessary csv files to run the other scripts. It requires the Final_Mohs_Billing.csv file in the data folder.


The merge_mohs_data.R script creates the Final_Mohs_Billing.csv file that is ran in get_surgeon_demographic_data.R. It requires all of the other files in the data folder.
